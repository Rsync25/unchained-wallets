Module: interaction

This module provides base classes for modeling interactions with keystores.

It also defines several constants used throughout the API for categorizing messages.

Integrations with new wallets should begin by creating a base class for that wallet by subclassing either DirectKeystoreInteraction or IndirectKeystoreInteraction.
